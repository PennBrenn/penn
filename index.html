Sponsor Booths · Swag & Discounts

💡 Codes are booth-only. Hit priority stops before 1 PM while traffic is light.

🔥 Priority — real value for your projects
DigiKey ⭐
Massive booth this year — XRP robots, edge AI demos, Meshtastic nodes, LED walls, PCB tile game. Actually interactive.
→ Stop by early. They run giveaways and hand out swag. Big booth = worth time.
OSH Cut ⭐
On-demand sheet metal laser cutting & bending. Next-day turnaround. Directly useful for enclosures.
→ Ask for a promo code — they distribute them at events like this.
PCBWay ⭐
PCB fab + assembly. Confirmed attending with "summer colors." Active first-order discounts online too.
→ Ask for show code. Also check pcbway.com for 1st-order deals ($5 off new accounts).
Prusa Research
The 3D printer brand. Likely showing hardware given the 3D printing panel happening today.
→ Ask about show specials / filament samples.
Arduino
New Arduino UNO Q (2GB RAM, 16GB eMMC, $59) will likely be on display at DigiKey or their own booth.
→ Stickers guaranteed. Ask about dev access / early programs.
Raspberry Pi
Confirmed sponsor. RPi 500+ and monitor in DigiKey booth. Own booth likely too.
→ Check for stickers / pins. Sometimes demo units on display.
Anthropic 👋
That's the company that makes me. Sponsored this year — first time at OpenSauce.
→ Worth a stop. May have Claude credits / swag. Ask what they're showing.
iFixit
Right-to-repair tools and teardowns. Their toolkit is genuinely useful for hardware work.
→ Stickers + possibly tool discounts. Ask at booth.
Framework
Modular repairable laptops. Dream hardware for someone who values repairability.
→ Ask about show discounts or demo unit pricing.
Cursor
AI coding editor. 50% off first month available via referral links online (not a booth-exclusive code).
→ Ask if they have a show-specific deal. Online: cursor.com referral links give 50% off month 1.
Also there — grab stickers / check it out
Kickstarter
Platform people. Stickers, maybe creator resources.
→ Quick stop.
Seeed Studio
Modules, XIAO boards, Grove ecosystem. Meshtastic nodes in DigiKey booth are Seeed.
→ Ask about show promos on XIAO / LoRa gear.
Elegoo
3D printers and Arduino-compatible boards. Budget-friendly maker gear.
→ Check for show deals / new product announces.
xTool
Desktop laser cutters. High likely they have a unit running live cuts.
→ If you've wanted a laser cutter — go ask about show pricing.
Fujifilm
Cameras + film. Unusual sponsor here — probably targeting creator/content side.
→ Instax freebies sometimes happen at these. Check it out.
DeVPN
Decentralized VPN. Privacy-focused product.
→ Ask about free tier / trial access.
Drover Labs
Maker tools / electronics. Not much public info yet.
→ See what they're showing.
Slingshot
Creator platform tools.
→ Free trial / early access common at events.
Fluke
Test & measurement gear (multimeters, clamp meters). Pro-grade tools.
→ Unlikely to give discounts but good to see what's new.
Formlabs
SLA/resin printers. High-res prints — relevant to fine enclosure work.
→ Ask about sample prints or material swatches.
Colorado PCB Assembly
PCB assembly house. Could be useful for future StudyPenn boards.
→ Grab a card, ask about quick-turn pricing.
GM / Wing / YouTube / Others
Brand sponsors. GM, Wing (drone delivery), YouTube — more marketing presence than maker goods.
→ Skip unless curious.
