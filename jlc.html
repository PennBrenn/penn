JLCPCB Basic Parts Cheat Sheet

All LCSC part numbers below are confirmed Basic library parts — no $3 extended fee. Always verify stock before ordering as availability can change.

💡 Best tool: Use jlcsearch.tscircuit.com — search by value, package, and filter by Basic. Much faster than the JLCPCB/EasyEDA interface.
💡 Backup strategy: Always note a primary and backup LCSC number. If primary is out of stock, swap to backup — same value, different brand, still Basic.
